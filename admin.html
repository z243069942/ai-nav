📋 如何获取 Token

  1. 访问 GitHub → Settings → Tokens → Generate new token (classic)
  2. 勾选 repo（完整仓库权限）
  3. 点击 Generate token，复制 token（只显示一次）
⚠️ Token 仅保存在浏览器 localStorage，不会上传任何服务器。
